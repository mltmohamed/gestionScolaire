# SchoolManage - Application de Gestion Scolaire\n\n## 🚀 Description\n\n**SchoolManage** est une application de bureau complète et moderne conçue pour simplifier la gestion quotidienne d\'un établissement scolaire. Face à la complexité du suivi des élèves, des enseignants, des finances et des classes, cette application offre une solution tout-en-un, centralisée et fonctionnant localement pour garantir la rapidité et la confidentialité des données.\n\nL\'objectif est de fournir une interface intuitive pour numériser et automatiser les tâches administratives, réduisant ainsi la charge de travail manuelle et les risques d\'erreur.\n\n## 🧠 Architecture\n\nL\'application est construite sur le framework **Electron**, ce qui lui permet de fonctionner comme une application de bureau native tout en utilisant les technologies web modernes. Son architecture se divise en deux processus principaux :\n\n-   **Processus Principal (Backend)** : C\'est le cœur de l\'application. Il s\'exécute en Node.js et gère la fenêtre de l\'application, les interactions avec le système d\'exploitation, et surtout, toute la logique métier. Il accède directement à la base de données SQLite pour toutes les opérations (lecture, écriture, mise à jour) et expose ces fonctionnalités de manière sécurisée au frontend via des canaux de communication IPC (Inter-Process Communication).\n\n-   **Processus de Rendu (Frontend)** : C\'est l\'interface utilisateur avec laquelle l\'administrateur interagit. Il s\'agit d\'une Single-Page Application (SPA) développée avec **React**. Elle est rapide, réactive et communique avec le processus principal pour demander ou envoyer des données, garantissant que l\'interface reste fluide sans jamais bloquer.\n\n## 🛠️ Technologies\n\nCe projet combine des technologies robustes et modernes pour offrir la meilleure expérience possible.\n\n-   **Framework Principal** : **Electron** - Pour créer des applications de bureau multiplateformes avec JavaScript, HTML et CSS.\n-   **Frontend** : **React** (avec Vite.js) - Pour une interface utilisateur dynamique et performante. Le routage est géré par `react-router-dom`.\n-   **Styling** : **Tailwind CSS** - Un framework CSS \"utility-first\" pour un design rapide, cohérent et personnalisable. Les icônes proviennent de `lucide-react`.\n-   **Base de Données** : **SQLite** (via `sql.js`) - Une base de données SQL légère, sans serveur, qui s\'exécute directement en JavaScript et sauvegarde les données dans un unique fichier (`schoolmanage.db`), rendant l\'application portable et autonome.\n-   **Gestion d\'État** : **React Hooks & Context API** - Pour une gestion d\'état simple et efficace, notamment pour l\'authentification et le thème.\n\n## 📸 Screenshots / Demo\n\n*\[Insérez ici des captures d\'écran de la page de connexion, du tableau de bord, de la gestion des élèves, etc.]*\n\n## ⚙️ Installation\n\nPour lancer le projet sur votre machine, suivez ces étapes :\n\n1.  **Clonez le dépôt**\n    ```bash\n    git clone https://github.com/votre-nom-utilisateur/app-gestion-scolaire.git\n    cd app-gestion-scolaire\n    ```\n\n2.  **Installez les dépendances**\n    Le projet utilise `npm` pour la gestion des paquets. Exécutez la commande suivante à la racine du projet :\n    ```bash\n    npm install\n    ```\n\n3.  **Lancez l\'application en mode développement**\n    Cette commande démarre simultanément le serveur de développement Vite pour le frontend et lance l\'application Electron.\n    ```bash\n    npm run electron:dev\n    ```\n    L\'application se lancera et vous pourrez vous connecter avec les identifiants par défaut : `admin` / `admin`.\n\n4.  **Compiler pour la production**\n    Pour créer un exécutable installable pour votre système d\'exploitation :\n    ```bash\n    npm run electron:build\n    ```\n\n## 📈 Résultat\n\nConcrètement, **SchoolManage** offre les fonctionnalités suivantes :\n\n-   **Authentification Sécurisée** : Une page de connexion protège l\'accès aux données de l\'application.\n-   **Tableau de Bord Dynamique** : Une vue d\'ensemble en temps réel des statistiques clés (nombre d\'élèves, enseignants, classes) et des activités récentes (dernières inscriptions, paiements).\n-   **Gestion Complète (CRUD)** :\n    -   **Élèves** : Ajout, modification, suppression et consultation des fiches élèves, avec photo de profil.\n    -   **Enseignants** : Gestion des informations des professeurs, incluant leur photo et spécialité.\n    -   **Classes** : Création des classes de la 1ère à la 9ème année et assignation d\'un professeur principal.\n-   **Module Financier Intégré** :\n    -   Suivi des paiements de **scolarité** et de **tenues scolaires**.\n    -   Gestion des **salaires** des enseignants.\n    -   Statistiques visuelles pour suivre les montants perçus et les soldes restants.\n-   **Persistance des Données** : Toutes les informations sont sauvegardées localement dans un fichier de base de données, garantissant que rien n\'est perdu entre les sessions.\n
